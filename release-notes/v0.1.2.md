# v0.1.2（Windows）

- 版本号对齐：`recorder_core` / `windows_collector` 升级到 0.1.2（用于 build-info 显示与自检）。
- Windows 侧功能无明显变化（本次主要是为发包与后续更新做准备）。

Android（开发中，未作为 Windows 主功能交付）：
- 修正按区间统计：避免每个 block 显示成整日同一份数据。
- 提升可读性：Top 列表改为“应用图标 + 多行”，Now 也显示图标。

下载与更新：
- 下载 `RecorderPhone-v0.1.2-windows.zip` 解压覆盖（推荐保留 `dist\\windows\\RecorderPhone\\data`）。
- 或使用应用内「Settings → Update」检查更新。
