# v0.1.1（Windows）

本次更新聚焦「更流畅 + 更不打扰」：

- 性能：Today 首页（Top / Flow / 0:00–24:00 泳道时间轴）做了缓存与降重建，Now 轮询刷新时明显更顺滑。
- 交互：Block 详情页不再额外拉取整日 timeline 做二次统计，打开更快。
- 通知：带 `Undo` 的提示条恢复为自动消失（避免一直挂在界面上）。
- Windows Toast：点击 `Skip / Pause / Resume` 改为后台执行，不再强制弹出/切换 UI；`Quick Review` 仍会正常打开 UI。

下载与更新：
- 直接下载 `RecorderPhone-v0.1.1-windows.zip` 解压覆盖（推荐保留 `dist\\windows\\RecorderPhone\\data`）。
- 或使用应用内「Settings → Update」检查更新。
