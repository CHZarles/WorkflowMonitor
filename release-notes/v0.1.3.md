# v0.1.3（Windows）

- Reports：点击日报/周报后，直接以“渲染后的 Markdown”展示（表格更易读，可选中复制）。
- 日报生成更稳：兼容带思考链/推理标签的 OpenAI-compatible 模型，自动提取最终内容并清理 `<think>/<analysis>`、代码围栏等噪音，避免“输出混入推理/拿不到结果”。
- 日报更有信息量：新增时间分布、碎片化（切换/上下文数）、黑名单时长、最长连续段等统计，默认日报 Prompt 也相应升级。

提示：
- 如果你之前改过 Prompt，默认 Prompt 的升级不会自动覆盖；可在 Reports 页面里点一次「Reset prompts to default」应用新版模板。

下载与更新：
- 下载 `RecorderPhone-v0.1.3-windows.zip` 解压覆盖（推荐保留 `dist\\windows\\RecorderPhone\\data`）。
- 或使用应用内「Settings → Update」检查更新。
